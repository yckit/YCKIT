使用说明：
----------------------------------------
1.把所有文件上传到服务器或者放置到本地服务器
2.如果是Linux服务器请用FTP把所有文件和文件夹权限修改成0777
3.在浏览器输入http://您的域名或者目录/install.php （注意：数据库如果没有权限建立数据库表请先手动建立数据库）
4.进入后台。(注意：默认帐号和密码都是admin)。
